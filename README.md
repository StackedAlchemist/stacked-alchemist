## 🛠 Tech Stack

All projects are built with:
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, responsive layouts
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Canvas API** — Games and charts
- **localStorage** — Client-side data persistence
- **External APIs** — Open Library, IGDB (via Cloudflare Worker proxy)

## 🚀 Running Locally

```bash
git clone https://github.com/StackedAlchemist/stacked-alchemist
cd Portfolio
# Open index.html in your browser
```

Most projects work by opening `index.html` directly. Projects using ES6 modules (Cosmic Rift Defender) require a local server:

```bash
npx serve .
# or use VS Code Live Server extension
```

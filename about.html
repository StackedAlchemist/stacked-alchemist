The Alchemist Behind the Code

Who I Am

My name is Billy Williams — though online I build under the name Stacked Alchemist. I'm a developer, founder, and lifelong builder based in Mesa, Arizona. I specialize in designing elegant software systems and tools that solve real-world problems.

To me, software development feels a lot like alchemy. You start with raw materials — ideas, problems, broken workflows — and through experimentation and craft you transform them into something powerful. Every project is a process of refinement: structure the system, shape the interface, and remove complexity until the result feels inevitable.

My focus is simple:

  • Clean architecture
  • Carefully crafted interfaces
  • Reliable systems that scale

No unnecessary complexity. Just well-forged software that works beautifully. My security background means I stay sharp, read the environment, and keep calm when things go sideways — which translates perfectly to debugging at 2am.

✦ Beyond the Code

The Builder Mentality

Outside of development I'm drawn to systems in many forms.

I grew up playing strategy and RPG games — StarCraft, Diablo, Final Fantasy, and World of Warcraft. Those games shaped how I think about systems, progression, and design. There's something about understanding how all the pieces fit together that never gets old.

I've also been building custom PCs since 2005 — choosing components, tuning performance, and creating machines that are both powerful and elegant. The same satisfaction I get from a clean system build carries directly into writing clean code.

And since 2011 I've been writing fantasy fiction, including my long-running project The Winglord Saga — a world where myth, strategy, and logic intersect.

Across all of these interests, one theme repeats itself: Build. Break. Refine. Repeat.
